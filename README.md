# Exercise 3 - Univariate Visualizations

This assignment explores univariate data visualization and sampling analysis using the EFI+ Mediterranean dataset.

## Visualizations Included
- Histograms
- Boxplots
- Strip plots
- Random sampling simulation

## Dataset
- EFIplus_medit.csv

## Tools Used
- Python
- pandas
- matplotlib
- seaborn
- Google Colab
- GitHub

## Assignment Objectives
- Explore univariate data distributions
- Compare visualization methods
- Analyze annual mean temperature (temp_ann)
- Understand sampling variability
